2.2 주석#

  • //
  • /* */
  • 논리적으로 정리하고 js의 틀에서 벗어나기 위해 주석을 먼저 작성하는 것이 낫다. 논리적으로 정리할 수 있는 능력이 프로그래밍보다 더 중요하다.
