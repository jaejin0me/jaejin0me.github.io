경계  - 박노해

과거를 팔아 오늘을 살지 말 것  
현실이 미래를 잡아먹지 말 것  
미래를 말하며 과거를 묻어버리거나  
미래를 내세워 오늘 할 일을 흐리지 말 것

순간의 감정 낭비를 하지 말 것  
하지만 숨기지 말고 솔직해질 것  
너무 우울한 생각 하지 말 것  
안심하여 늘어지지도 말 것

이 모든 것을 언제나 새길 것
