자기 자신을 비관하지 말 것  
자기 자신을 낮추지 말고 자신을 사랑할 것  

다른 사람이랑 비교하지 말 것  
자기 자신에게 항상 되물을 것

최선을 다해서 공부할 것  
긍정적으로 생각할 것  
많은걸 추구하지 말고 더 가치 있고  
좋은 걸 추구할 것


하고 싶은걸 찾을 것  
남의 눈 신경 쓰지 말고 일단 도전할 것  
하고 싶은 거 꼭 할 것  
후회하지 않을 정도로 하고 싶은 거라면  
정말 늦기 전에 꼭 해야 된다는 것

익숙함에 속아 소중함을 잃지 말 것  
기회가 오면 놓치지 말고 꼭 잡을 것  
위를 보고 살 것, 또한 아래를 보고 살 것

포기하면 편하다 하지만 순간의  
편안함을 위해 포기하지 말 것  
남 탓하지 말 것, 변명하지 말 것, 정직하게 살 것  
땀 없인 달콤함도 없을 것

남 걱정하지 말고 너나 잘할 것  
연예인 걱정이 가장 쓸데없는 걱정일 것  
숨 쉬는 한 희망은 있을 것, 뭐든지 즐겁게 할 것  
보다 세상을 넓게 볼 것, 사람을 너무 믿지 말 것  
감정에 솔직할 것
